# Assets Directory

Place your add-in icons here:

- `icon-16.png` - 16x16 icon for ribbon
- `icon-32.png` - 32x32 icon for manifest
- `icon-64.png` - 64x64 high resolution icon
- `icon-80.png` - 80x80 icon for ribbon

## Icon Requirements

- PNG format
- Transparent background
- Clear, simple design
- Represents Nubia AI branding

## Placeholder Icons

For development, you can use placeholder icons or replace the manifest URLs with data URIs for simple colored squares.
