# Nubia - Quick Start Guide

## Installation

### Windows
1. Download `Nubia-Setup-1.0.0.exe` from the dist folder
2. Run the installer as Administrator
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut

### Mac
1. Download `Nubia-1.0.0.dmg` from the dist folder
2. Double-click to mount the disk image
3. Drag Nubia to Applications folder
4. Launch from Applications (may need to allow in Security & Privacy settings)

### Linux
1. Download `Nubia-1.0.0.AppImage` from the dist folder
2. Make executable: `chmod +x Nubia-1.0.0.AppImage`
3. Run: `./Nubia-1.0.0.AppImage`

## First Run Setup

1. **Welcome Screen**: Click "Get Started" to begin onboarding
2. **Excel Detection**: The app will check if Excel is installed
3. **API Configuration**: 
   - Choose your AI provider (OpenAI or Anthropic)
   - Enter your API key
   - Test the connection
4. **Sample Tasks**: Try the provided examples to get familiar

## API Key Setup

### OpenAI
1. Visit https://platform.openai.com/api-keys
2. Create a new API key
3. Copy and paste into Nubia settings
4. Recommended models: GPT-4, GPT-3.5-turbo

### Anthropic
1. Visit https://console.anthropic.com/
2. Generate an API key
3. Copy and paste into Nubia settings
4. Recommended models: Claude-3-opus, Claude-3-sonnet

## Chat Interface Tips

1. **Be Specific**: "Create a budget spreadsheet with 12 months" vs "Make a budget"
2. **Context Matters**: Mention which worksheet or data range you're working with
3. **Step by Step**: Break complex tasks into smaller requests
4. **Review Results**: Always check the generated Excel content before saving

## Troubleshooting

### Excel Not Detected
- Ensure Microsoft Excel is installed and can open normally
- Try restarting both Excel and Nubia
- Check that Excel isn't running in compatibility mode

### API Connection Issues
- Verify your API key is correct and active
- Check your internet connection
- Ensure you have sufficient API credits/quota
- Try switching between OpenAI and Anthropic

### Performance Issues
- Close other Excel workbooks before using Nubia
- Restart the application if it becomes unresponsive
- Check Task Manager for high memory usage

### Automation Not Working
- Make sure Excel is visible (not minimized)
- Disable Excel add-ins that might conflict
- Check Windows security settings for automation permissions

## Keyboard Shortcuts

- `Ctrl+N` (Win) / `Cmd+N` (Mac): New chat
- `Ctrl+Enter` (Win) / `Cmd+Enter` (Mac): Send message
- `Ctrl+,` (Win) / `Cmd+,` (Mac): Open settings
- `F1`: Open help dialog
- `Escape`: Cancel current operation
